В приложении Den внеси следующие изменения:
1. Случайные вариации позитивного и негативного вопросов
Замени фиксированные вопросы "Что рассмешило?" и "Что раздражало?" на случайные вариации.
Массив позитивных вопросов:

"Что рассмешило?"
"Что порадовало?"
"Что было приятным?"
"Что вдохновило?"
"Что удивило в хорошем смысле?"
"За что благодарен сегодня?"

Массив негативных вопросов:

"Что раздражало?"
"Что расстроило?"
"Что было сложным?"
"Что заставило понервничать?"
"Что хотелось бы изменить?"

Вопрос выбирается случайно один раз при создании новой записи и сохраняется в структуре DayEntry. При просмотре старых записей показывается та формулировка которая была выбрана в тот день.
Обнови структуру DayEntry в storage.ts:
javascriptanswers: {
  learned: string,
  met: string,
  positive: {
    question: string, // выбранная формулировка
    answer: string,
    category: "positive"
  },
  negative: {
    question: string,
    answer: string,
    category: "negative"
  },
  dayQuestion: string,
  proud: string,
  notes: string
}
Сохрани обратную совместимость — старые записи с полями laughed и annoyed продолжают отображаться корректно.
2. Отключение напоминаний
В SettingsScreen добавь переключатель (Switch) "Напоминания" в раздел уведомлений:

По умолчанию включены
При выключении — отменяй все запланированные уведомления через expo-notifications
При включении — планируй уведомление на выбранное время
Если напоминания выключены — TimePicker скрывается или становится неактивным
Сохраняй состояние переключателя в AsyncStorage как 'notifications_enabled'

3. Исправить полные фото в режиме просмотра
В режиме просмотра дня фотографии должны открываться на полный экран при нажатии на миниатюру. Добавь модальное окно с полноэкранным просмотром фото, кнопкой закрытия и возможностью листать между фото свайпом.
4. Экспорт дня как картинка
Добавь кнопку "Поделиться днём" на экране просмотра записи дня.
При нажатии генерируется красивая карточка с:

Датой и днём недели крупно вверху
Эмодзи настроения
Всеми заполненными ответами (пустые не показываются)
Логотипом "Den" внизу мелко

Карточка сохраняется как изображение и открывается стандартный шаринг Android. Используй react-native-view-shot для захвата экрана или генерируй SVG/Canvas.
